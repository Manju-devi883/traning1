# Day 2 - Python Basics, Git & GitHub

## Objective
The objective of this session was to understand the fundamentals of Python programming and learn how Git and GitHub are used for version control and project management.

## Topics Covered

### Python Basics
- Introduction to Python
- Features of Python
- Variables and Data Types
- Input and Output
- Operators
- Conditional Statements (`if`, `else`)
- Loops (`for`, `while`)
- Functions
- Basic File Handling

### Git Basics
- What is Git?
- Why Git is used
- Git Installation
- Git Configuration
- Creating a Local Repository
- Basic Git Commands
  - `git init`
  - `git status`
  - `git add`
  - `git commit`
  - `git log`

### GitHub
- Introduction to GitHub
- Creating a GitHub Account
- Creating a New Repository
- Connecting Local Repository with GitHub
- Pushing Code to GitHub

### Development Environment
- Installing Visual Studio Code
- Setting up Python Environment
- Using the VS Code Terminal

## Hands-on Practice
- Installed Python, Git, and Visual Studio Code.
- Created a local Git repository.
- Learned basic Git commands.
- Created a GitHub repository.
- Connected the local project to GitHub.
- Successfully pushed the first project to GitHub.

## Key Learnings
- Learned the basics of Python programming.
- Understood the importance of version control using Git.
- Learned how GitHub is used to store and manage projects.
- Gained hands-on experience with Git commands and repository management.

## Summary
Day 2 focused on building the foundation required for AI development. Python was introduced as the primary programming language, while Git and GitHub were used to understand version control, project management, and code collaboration. Practical exercises included setting up the development environment, creating repositories, and pushing code to GitHub.
