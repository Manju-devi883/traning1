# Day 4 - Playwright, Vibe Coding & Context Engineering

## Objective
The objective of this session was to understand modern AI development practices, browser automation using Playwright, AI-assisted development through Vibe Coding, and the importance of Context Engineering in AI applications.

---

# 1. Playwright

## Theory
Playwright is an open-source browser automation library that allows developers to automate web browsers such as Chrome, Firefox, and Safari. It is widely used for web testing, web scraping, and browser automation.

### Example
A Playwright script can automatically open Google, search for "Artificial Intelligence", and capture a screenshot of the search results without any manual interaction.

---

# 2. Vibe Coding

## Theory
Vibe Coding is an AI-assisted software development approach where developers use AI tools to generate code, debug programs, explain concepts, and improve productivity. Instead of writing every line manually, developers collaborate with AI to build applications faster.

### Example
A developer asks ChatGPT:
> "Create a Python calculator application."

The AI generates the initial code, and the developer reviews, tests, and improves it before using it in the project.

---

# 3. Context Engineering

## Theory
Context Engineering is the process of providing the right information, instructions, and relevant data to an AI model so it can generate more accurate and useful responses. Good context leads to better AI performance.

### Example
Instead of asking:
> "Write code"

Provide proper context:
> "Write a Python program to read a CSV file, calculate the average marks of students, and display the result."

The second prompt gives the AI enough context to produce a more accurate solution.

# Key Learnings

- Learned how Playwright automates browser tasks.
- Understood how AI can improve software development through Vibe Coding.
- Learned that providing better context results in better AI responses.
